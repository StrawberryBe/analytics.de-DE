# Erstellen einer Data Warehouse-Anforderung

In diesen Schritten wird beschrieben, wie Sie eine Data Warehouse-Anforderung erstellen.

1. Navigieren Sie zu **[!UICONTROL Analytics]** > **[!UICONTROL Tools]** > **[!UICONTROL Data Warehouse]**.
1. On the [!UICONTROL Data Warehouse Request] tab, configure the options described in [Data Warehouse Requests Descriptions](/help/export/data-warehouse/data-warehouse.md#section_F21C78ED36884C389C852E876AF5CDE8)
1. Klicken Sie auf **[!UICONTROL Request this Report]**.
